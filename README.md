# Gatarhero-game

Static GitHub Pages site that lets you import a song, record a chart by tapping keys, and play it back.

## Use
- Open `index.html` locally or via GitHub Pages.
- Load a song file (mp3/wav/ogg).
- Click **Record chart** and tap A S D F G to the beat.
- Click **Play chart** to test.
- Export/Import charts as JSON for sharing.

## GitHub Pages
- Push to a repo and enable Pages for the `main` branch / root.
